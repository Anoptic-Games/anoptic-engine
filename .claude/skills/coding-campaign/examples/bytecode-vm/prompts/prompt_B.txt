    enum Op { Push(i64), Pop, Add, Sub, Mul, Div, Mod, Neg, Dup, Swap, Jmp(usize), Jz(usize), Print, Halt }

Define exactly (it MUST be `pub`):

    pub fn run(program: &[Op]) -> Result<Vec<i64>, String>

Semantics: maintain a Vec<i64> operand stack and a program counter starting at 0; loop executing program[pc].
- Push(n): push n; pc+=1.
- Pop: discard top; pc+=1.
- Add/Sub/Mul/Div/Mod: pop b then pop a, push (a op b); pc+=1. Use i64 truncating `/` and `%`. Division or modulo by zero returns Err. Prefer wrapping arithmetic so overflow never panics.
- Neg: pop a, push -a; pc+=1.
- Dup: push a copy of the current top; pc+=1.
- Swap: swap the top two; pc+=1.
- Jmp(t): set pc=t (do not increment).
- Jz(t): pop a; if a==0 set pc=t else pc+=1.
- Print: pop a and append it to the output vector; pc+=1.
- Halt: stop and return Ok(output).
Any stack underflow, division/modulo by zero, or a program counter that runs past the end of the program (without hitting Halt) returns Err with a short message. Return Ok(outputs) only on Halt.
